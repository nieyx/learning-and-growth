### 浮动

#### 什么是浮动
```
当一个盒子使用了float的属性时，导致父级盒子不能被撑开，这种现象叫做浮动
```
#### 浮动产生的原因
```
+ 当盒子使用的float属性，float属性具有脱离文档流的特性
+ 当父级盒子没有设置高度时，导致盒子无法被撑开，
```

#### 浮动产生的副作用
```
+ 背景不能够显示
+ 边框无法正常显示
+ margin／padding无法正常显示
```
#### 清除浮动的几种方法
```
+ 为使用float的元素的父元素设置高度
+ 在使用float的元素后，加上一个空的div，添加一个clear：both的样式
+ 使用after的伪类
```
